# 益利肉類食品販售網站 - 專案總結

## 專案概述

這是一個完整的肉類食品販售網站，包含前台購物功能和後台管理系統。

## 技術架構

### 前端
- **框架**: React 18.2
- **路由**: React Router 6
- **狀態管理**: React Context API
- **HTTP 客戶端**: Axios
- **樣式**: CSS3 (響應式設計)

### 後端
- **框架**: Node.js + Express
- **資料庫**: MySQL
- **認證**: JWT (JSON Web Token)
- **檔案上傳**: Multer
- **Email**: Nodemailer (Gmail SMTP)

## 資料庫結構

### 資料表

1. **categories** - 商品分類
2. **products** - 商品資訊
3. **orders** - 訂單主檔
4. **order_items** - 訂單明細
5. **admins** - 管理員帳號

## API 端點

### 前台 API
- `GET /api/products` - 取得商品列表（可選分類篩選）
- `GET /api/products/:id` - 取得商品詳情
- `GET /api/categories` - 取得分類列表
- `POST /api/orders` - 建立訂單

### 後台 API
- `POST /api/auth/login` - 管理員登入
- `GET /api/products/admin/all` - 取得所有商品（管理用）
- `POST /api/products/admin` - 新增商品
- `PUT /api/products/admin/:id` - 更新商品
- `DELETE /api/products/admin/:id` - 刪除商品

3. 使用 PM2 管理 Node.js 進程
4. 配置 Nginx 反向代理
5. 啟用 HTTPS
6. 設定資料庫備份
