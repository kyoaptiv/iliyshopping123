# 如何進入後台管理系統

## 🚪 進入後台的方法

### 方法一：直接訪問登入頁面（最簡單）

1. **確保網站正在運行**
   - 前端應該運行在：http://localhost:3000

2. **在瀏覽器輸入以下網址**：
   ```
   http://localhost:3000/admin/login
   ```

3. **輸入登入資訊**：
   - **帳號**：`admin`
   - **密碼**：`12345`

4. **點擊「登入」按鈕**

5. **成功後會自動跳轉到管理後台**

---

### 方法二：從網站首頁導航（如果已建立連結）

如果網站有後台入口連結，可以直接點擊進入。

## 🔧 如果無法登入

### 問題 1：找不到登入頁面

**症狀**：訪問 http://localhost:3000/admin/login 顯示 404 錯誤

**解決方法**：
1. 確認前端伺服器正在運行
2. 確認路由設定正確
3. 檢查瀏覽器網址是否正確

### 問題 2：登入失敗

**症狀**：輸入帳號密碼後顯示錯誤訊息

**檢查步驟**：

1. **確認後端正在運行**
   - 訪問：http://localhost:5000/api/health
   - 應該看到成功訊息

2. **確認資料庫連線正常**
   - 執行 `檢查設定.bat`
   - 或執行 `使用XAMPP MySQL.bat` → 選擇 [4] 檢查管理員帳號

3. **確認管理員帳號存在**
   ```sql
   USE yili_meat_shop;
   SELECT * FROM admins WHERE username='admin';
   ```
   應該看到一筆資料，密碼欄位應該是 `12345`

4. **檢查 .env 檔案**
   - 確認 `server/.env` 檔案存在
   - 確認資料庫設定正確

5. **查看後端控制台**
   - 查看是否有錯誤訊息
   - 檢查登入請求是否成功

### 問題 3：登入後無法訪問後台

**症狀**：登入成功但無法進入管理頁面

**解決方法**：
1. 檢查瀏覽器開發者工具（F12）→ Console 標籤
2. 查看是否有 JavaScript 錯誤
3. 確認後端 API 可以正常訪問
